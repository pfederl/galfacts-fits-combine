Detailed Description

contains representation of an n-dimensional slice, which is a structure used to create a view of an array

this is inspired by the NumPy basic slicing strategy

Some examples of usage:

NumPy: [3::-1,:,3] C++ : SliceND s; s.start(3).step(-1) .next() .next().index(3); or SliceND s; s.axis(0).start(3).step(-1); s.axis(2).index(3); or typedef Slice1D S; SliceND s({ S().start(3).step(-1), S(), S(3) });
